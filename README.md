# AI-Powered Clinical Trial Eligibility & Matching Engine

## 🏥 Healthcare Innovation Challenge
An intelligent system that analyzes anonymized patient health records and automatically matches them to suitable clinical trials using AI and ML techniques.

## 🚀 Quick Start

## 🏗️ System Architecture

### AI Matching Engine (Python + Flask)
```bash
cd python-api
pip install -r requirements.txt
python app.py
```

### Frontend Dashboard (React + Vite)
```bash
cd frontend
npm install
npm run dev
```

### Data Processing Backend (Node.js + Express)
```bash
cd backend
npm install
npm start
```

## Project Structure
```
├── frontend/          # React + Vite frontend
├── backend/           # Node.js + Express API
├── python-api/        # Python Flask API
├── nextjs-app/        # Full-stack Next.js app
├── shared/            # Shared utilities and types
├── docs/              # Documentation
└── scripts/           # Build and deployment scripts
```

## Quick Commands
- `npm run setup` - Install all dependencies
- `npm run dev` - Start all development servers
- `npm run build` - Build all projects
- `npm run deploy` - Deploy to production

## Environment Setup
Copy `.env.example` to `.env` and fill in your configuration.
